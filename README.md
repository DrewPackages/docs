# Drew Documentation

This is documentation for [Drew](https://drewpacakges.com/) dApp package manager.

## Development

Setup dependencies:
```
bun install
```
Run development server:
```
bun docs:dev
```
On push to master branch the documentation is automatically deployed to [Drew Docs page](https://docs.drewpacakges.com/)
